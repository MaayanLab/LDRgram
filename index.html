LDRgram

The LINCS Dataset Registry (LDR) is a DCIC web application built to allow the uploading and approval of experimental data provided by LINCS Data Signature Generation Centers. The clustergram shows the number of perturbations applied to cell lines for each assay. Blue tiles represent released data and orange tiles represent unreleased data and some tiles are split to represent partially released data.

LDRgram is being developed by the Ma'ayan Lab at the Icahn School of Medicine at Mount Sinai.
